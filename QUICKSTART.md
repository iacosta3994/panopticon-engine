# Panopticon Engine - Quick Start

## 🚀 Deploy in 2 Minutes

### For Ian (or anyone):

**Step 1**: Fork this repo (click Fork button above)

**Step 2**: Go to [vercel.com](https://vercel.com) → Add New Project → Import your fork

**Step 3**: Add environment variable:
```
JWT_SECRET = any-random-string-here
```

**Step 4**: Click Deploy

**Step 5**: Visit `your-app.vercel.app/dashboard` ✅

## 🔑 Want Alerts?

### Email Alerts
Add to Vercel env vars:
```
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=your-app-password
```

### Telegram Alerts  
```
TELEGRAM_BOT_TOKEN=your-token
TELEGRAM_CHAT_ID=your-chat-id
```

**That's it!** Alerts work immediately.
